# ktav-lang

**Languages:** [English](README.md) · **Русский** · [简体中文](README.zh.md)

> **Ktav** (כְּתָב, *«письмо»*) — простой формат конфигурации.
> Форма JSON, без кавычек, ключи через точку, многострочные
> строки «из коробки». Правила умещаются на одном экране;
> никаких ловушек с отступами.

## На вкус

```text
port: 20082

banned_patterns: [
    .*\.onion:\d+
]

upstreams: [
    {
        host: a.example
        port: 1080
        timeouts: {
            read: 30
            write: 10
        }
    }
]

motd: (
    Welcome to the node.
    Please behave.
)
```
