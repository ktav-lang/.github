# Code of Conduct

The `ktav-lang` organisation adopts the **[Contributor Covenant,
version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
in full. The canonical text lives at the link above; we do not
maintain a local fork of it.

## Scope

This code applies to every space operated by the organisation:
GitHub issues and pull requests, discussions, repository code and
commit messages, and any official communication channel.

## Reporting

To report an incident, email **phpcraftdream@gmail.com**. Reports
are handled in private. Retaliation against a reporter is itself a
violation.

## Enforcement

Maintainers apply the enforcement ladder described in the Covenant
text (Correction → Warning → Temporary Ban → Permanent Ban). The
response is calibrated to severity and to the contributor's history
in the organisation.
